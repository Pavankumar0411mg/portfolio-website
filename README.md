# Personal Portfolio Website

A modern, responsive portfolio website showcasing my skills, projects, and contact information. Built with HTML, CSS, and JavaScript with dynamic GitHub integration.

## Features

- **Responsive Design** - Works perfectly on all devices
- **GitHub Integration** - Automatically displays latest repositories
- **Modern UI/UX** - Clean and professional design
- **Contact Form** - Easy way for recruiters to get in touch
- **Resume Integration** - Direct link to view/download resume

## Sections

- **Profile** - Introduction and key highlights
- **About Me** - Background and career objectives
- **Skills** - Technical skills with visual cards
- **Projects** - Dynamic GitHub repository showcase
- **Contact** - Contact information and form

## Technologies Used

- HTML5
- CSS3 (Flexbox, Grid, Animations)
- JavaScript (ES6+)
- GitHub API
- Responsive Design
